## Run Locally

Clone the project

```bash
 git clone https://github.com/Asif-rehman012/Price_Comparison_App.git
 cd Price_Comparison_App

```

Backup setup

```bash
cd backend
npm install
node server.js

```
Frontend Setup

```bash
cd frontend
npm install
npm run dev

```

Frontend runs on :

```bash
  http://localhost:5173

```
